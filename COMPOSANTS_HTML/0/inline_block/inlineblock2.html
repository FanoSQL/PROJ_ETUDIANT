C'est quoi une balise inline ?

en HTML 5 et CSS 3 inline : Est une balise qui se trouve obligatoirement à l'intérieur d'une balise block.
Une balise inline ne crée pas de retour à la ligne, le texte qui se trouve à l'intérieur s'écrit donc à la suite du texte précédent, sur la même ligne (c'est pour cela que l'on parle de balise « en ligne »).

span, a href, button, input sont quelques balises inline.
Elles peuvent donc être contenues dans des balises de block !
